## ğŸ“ Description

CheckList is a modern, dark-themed web application built with React and Tailwind CSS. It offers a clean, grid-based interface for tracking habits and daily tasks over a full calendar year.

### Key Features:

*   **Monthly Grid View:** Visualize your progress for the entire month at a glance.
*   **12-Month Navigation:** Easily switch between months using the month list or calendar controls.
*   **Task Management:**
    *   **Add Tasks:** Create new tracking items with the green "Add Task" button.
    *   **Edit Tasks:** Rename tasks inline using the yellow edit icon.
    *   **Delete Tasks:** Remove items using the red close icon.
*   **Interactive Tracking:**
    *   Click once to mark as **Done** (Green Check).
    *   Click twice to mark as **Missed** (Red X).
    *   Click again to reset.
*   **Smart Indicators:**
    *   **Current Day:** Automatically highlighted with a bold green triangle pointer.
    *   **Scoring:** Real-time percentage score calculated for each task based on completion.
*   **Dark UI:** A polished "Neutral-950" dark theme with high-contrast colors (Green, Light Blue, Yellow) for optimal readability.
